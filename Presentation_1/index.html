Как оно работает?

Машина генерации нового токена

Токенизация

Процесс разбиения текста на маленькие части (токены), которые модель может обработать

Контекстное окно

  • Ограниченная "память" модели при обработке текста
  • Размер окна = макс. длина последовательности токенов
  • Примеры:
    • GigaChat: 32K токенов
    • GPT-4o: 128K токенов
    • Claude 3.5: 200K токенов

Промптинг и Взаимодействие

In-context Learning

Способность модели учиться на примерах прямо в контексте диалога без дополнительного обучения

Prompt Engineering

Искусство составления эффективных запросов к ИИ

Базовые Принципы

  • Четкость и конкретность
  • Контекст и детали
  • Пошаговые инструкции
  • Указание формата ответа

Практический Пример

❌ Плохой промпт: 
Сгенерируй SQL скрипт, чтобы найти пользователей, которые поставили низкую оценку
✅ Хороший промпт: 
РОЛЬ: Ты - интерфейс postgres, преобразующий запросы естественного языка в SQL скрипты
                            
СХЕМА ТАБЛИЦЫ:
USER_ID(строка),
REVEW_RATING(целочисленное 0-5),
CREATED_AT(таймстемп),
USER_NAME(строка)

ФОРМАТ ВЫВОДА: Корректный SQL скрипт для postgres, без комментариев

ЗАПРОС: Найди пользователей, которые поставили низкую оценку, низкая оценка это 0, 1, 2
